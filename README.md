## 기능 설명

### 캐릭터 검색 및 등록

검색: 검색창에 캐릭터명을 입력 후 버튼을 눌러 캐릭터를 검색.
등록: 검색된 캐릭터를 저장하여 관리 가능.
선택 및 삭제: 캐릭터별로 보스를 관리하거나 캐릭터를 삭제 가능.

### 주간 보스 관리

보스 이미지를 클릭하여 선택/해제.
난이도와 파티원 수를 조정.
현재 선택한 보스의 결정석 가격을 실시간으로 계산.

### 총 결정석 가격

각 캐릭터의 결정석 가격 총합 표시.
모든 캐릭터의 총 결정석 가격 합산 표시.
